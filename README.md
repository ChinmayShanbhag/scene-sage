# SceneSage

A multimodal video scene analysis and trailer recommendation system. SceneSage extracts frames from movie trailers, encodes them using OpenCLIP and Sentence Transformers into 896-dimensional vectors, and indexes them with FAISS for fast similarity search.

Given an image, video clip, or YouTube link, SceneSage finds the most visually similar trailers from its index.

## How It Works

1. **Frame Extraction** — Trailers are downloaded via `yt-dlp` and sampled at 1 fps using FFmpeg
2. **Visual Encoding** — Each frame is encoded with OpenCLIP (`ViT-B-32-quickgelu`, OpenAI weights) into a 512-dim vector
3. **Text Encoding** — Subtitle segments are encoded with Sentence Transformers (`all-MiniLM-L6-v2`) into 384-dim vectors
4. **Vector Fusion** — Visual and text vectors are concatenated into 896-dim scene vectors
5. **Indexing** — All vectors are L2-normalized and stored in a FAISS `IndexFlatIP` for cosine similarity search
6. **Retrieval** — Query vectors are matched against the index, then deduplicated and ranked at the trailer level

## Getting Started

### Prerequisites

- Python 3.10+
- FFmpeg (must be on `PATH`)
- `yt-dlp` (installed via `pip` or standalone)

### Installation

```bash
git clone https://github.com/ChinmayShanbhag/scene-sage.git
cd scene-sage
pip install -r requirements.txt
```

### Building the Index

Run the Jupyter notebook end-to-end to download trailers, extract frames, and build the FAISS index:

```bash
jupyter notebook scene_sage_baseline.ipynb
```

This generates the `artifacts/` directory containing the search index and metadata.

### Running the Web Demo

The web demo lets you upload an image, a video, or paste a YouTube URL to get trailer recommendations from the indexed data.

```bash
python web/server.py
```

Then open [http://localhost:8090](http://localhost:8090) in your browser.

> **Note:** The demo requires the `artifacts/` directory to exist. Run the notebook first to generate it. Models are loaded lazily on the first search request.

### Upload Limits

| Input Type | Max Size |
|------------|----------|
| Image      | 10 MB    |
| Video      | 50 MB    |

Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.bmp` (images) and `.mp4`, `.webm`, `.mkv`, `.avi`, `.mov` (videos).

## Tech Stack

| Component | Library |
|-----------|---------|
| Visual Encoder | OpenCLIP (`ViT-B-32-quickgelu`) |
| Text Encoder | Sentence Transformers (`all-MiniLM-L6-v2`) |
| Vector Index | FAISS |
| Frame Extraction | FFmpeg, OpenCV |
| Video Download | yt-dlp |
| Web Backend | FastAPI + Uvicorn |
| Frontend | Vanilla HTML/CSS/JS |
